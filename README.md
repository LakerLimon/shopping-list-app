# Shopping List App – Список покупок с общим доступом

## О проекте

Проект представляет собой мобильное приложение и веб-версию для совместного ведения списков покупок. Разработка выполняется в рамках дисциплины «Системная и программная инженерия».

## Lean-подход

Проект реализуется с использованием методологии Lean. На первом этапе разработана и развёрнута веб-версия, которая выступает в роли минимально жизнеспособного продукта. Это позволяет проверить гипотезу о востребованности совместных списков покупок минимальными средствами. Мобильное приложение будет разработано только после подтверждения гипотезы на основе анализа использования веб-версии.

## Структура репозитория

Корневая директория содержит следующие папки и файлы.

**shopping-list-mvp** – веб-версия (MVP). Включает серверную часть на Django, модуль регистрации и авторизации, основной модуль списка покупок, статические файлы, HTML-шаблоны, Dockerfile для контейнеризации, управляющий скрипт manage.py и файл зависимостей requirements.txt.

**mobile-app** – директория для мобильного приложения. На данный момент находится в стадии планирования, разработка не начата.

**docker-compose.yml** – файл для оркестрации контейнеров веб-версии.

**README.md** – описание проекта.

## Веб-версия (MVP)

Статус: разработана и развёрнута.

Веб-версия включает минимальный функционал для проверки гипотезы. Доступны регистрация и авторизация пользователей, создание списка покупок, добавление товаров в список, отметка товаров как купленных и приглашение участников по ссылке.

Технологии веб-версии: Python, Django, SQLite на этапе разработки и PostgreSQL на этапе развёртывания, Docker и Docker Compose для контейнеризации, HTML и CSS для интерфейса.

## Мобильное приложение

Статус: в стадии планирования.

Разработка мобильного приложения начнётся после подтверждения гипотезы. Планируемый функционал включает всё, что уже реализовано в веб-версии, а также push-уведомления, историю покупок, категории товаров, приглашения по QR-коду и офлайн-режим с локальным кэшированием.

Планируемые технологии: операционная система Android, язык Java, среда разработки Android Studio, сервис Firebase Cloud Messaging для уведомлений, библиотека Room Database для локального хранения данных.
